# Vanilla Node.js REST API | No Framework used

> Full CRUD REST API using Node.js with no framework

## Routes

```bash
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id

```

## Installation

```bash
# Install dependencies
npm install

# Run in develpment
npm run dev

# Run in production
npm start
```
